What is HTML?

HTML (hyperlink text markup language) is a markup language that defines the structure of the content. It is not a programming language.

What is CSS?

CSS (cascading style sheets) is a style sheet language, neither programming language, nor markup language.

What is Javascript?

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

Javascript Basics

Variables are containers that you can store values in. variables are needed to do anything interesting in programming. If values couldn't change, then you couldn't do anything dynamic.
